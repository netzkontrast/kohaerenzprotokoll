# Structure Methods

Plot-Strukturen + Storyform-Build-Loop + Validation, selektierbar in
`intent.methods_preference.structure` (Plot-Strukturen mehrere kombinierbar
— z.B. `40-chapter-matrix + dramatica-quad`).

## Plot-Strukturen (Phase 2 + 5)

| File | Methode | Wann verwenden |
|------|---------|----------------|
| [`40-chapter-matrix.md`](./40-chapter-matrix.md) | 4 Akte × 10 Kapitel | Default für Romane ab 80k Wörter |
| [`heroes-journey.md`](./heroes-journey.md) | 12 Stufen (Campbell/Vogler) | Fantasy, Coming-of-Age, mythische Romane |
| [`save-the-cat.md`](./save-the-cat.md) | 15 Beats (Snyder) | Genre-Romane (Thriller, Romance) |
| [`dramatica-quad.md`](./dramatica-quad.md) | Fraktale Quad-Rekursion | In Kombination mit anderen — gibt Theorie-Stütze |

Phase 5 lädt diese on demand. Bei mehreren Auswahlen wird die Kombination
formuliert (z.B. „40-Kapitel-Matrix mit Dramatica-Quad pro Kapitel").
