# Examples

Worked Examples für die Phasen-Outputs. **Bewusst NICHT projekt-spezifisch**
(kein Kohärenz-Protokoll), sondern generische Beispiele zum Verstehen der
Schema-Strukturen.

| File | Phase | Beispiel-Projekt |
|------|-------|------------------|
| `example-intent.yaml` | 1 | Hypothetisches "consciousness-novel" (Hard-SF mit philosophy-as-engine) |
| `example-architecture.yaml` | 2 | Single storyform für consciousness-novel |

## Zweck

- **Schema-Validation:** Zeigt strukturell, wie ein vollständiger Phase-Output aussieht
- **Lern-Material:** User können sich am Beispiel orientieren
- **Demo-Mode-Quelle:** `bootstrap_project.sh` kann Demo-Projekt aus diesen Examples erstellen

## Wichtig

Diese Examples sind NICHT zum Editieren — sie sind Read-Only Referenz. Beim
Anlegen eines neuen Projekts werden die Templates aus `assets/` kopiert,
nicht die Examples.

## Erweiterbar

Weitere Examples (z.B. Dual-Storyform für Kohärenz-Pattern, Fantasy-Beispiel,
Literary-Beispiel) können hier ergänzt werden.
