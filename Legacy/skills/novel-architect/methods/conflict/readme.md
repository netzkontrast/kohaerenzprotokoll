# Conflict Engine Methods

Treibende Konflikt-Mechaniken — nicht *was* der Konflikt ist (das ist in
`intent.core_conflict_question`), sondern *wie* er strukturell wirkt.

| File | Methode | Wann verwenden |
|------|---------|----------------|
| `philosophy-as-engine.md` | Philosophische Positionen treiben Plot | Hard-SF mit Theoriebezug, Literary Fiction mit Ideen-Plots |
| `science-as-engine.md` | Wissenschaftliche Theorien als Konflikt-Treiber | Hard-SF, Cli-Fi, Bio-Thriller |
| `dual-storyform.md` | Zwei parallele Storyforms (5D-Interferenz) | Advanced, mehrdeutige Narrative, philosophischer Anspruch |

Phase 2 lädt diese on demand basierend auf
`intent.methods_preference.conflict`. Mehrere kombinierbar.
