# Character Methods

Selektierbar pro Projekt via `intent.methods_preference.character`. Mehrere
Modelle pro Charakter möglich (primary + secondary).

| File | Modell | Wann verwenden |
|------|--------|----------------|
| `tsdp-ifs.md` | Tertiäre Strukturelle Dissoziation + IFS | Fragmentierte Identität, Trauma-Roman |
| `big-five.md` | OCEAN | Realistische Hauptfiguren, Drama |
| `enneagramm.md` | 9 Typen + Wings + Integration/Disintegration | Klar motivierte Charaktere, Ensembles |
| `jung-archetypes.md` | Archetypen + Individuation | Mythische, Coming-of-Age, Fantasy |

Schema-Detail jeweils im File. Phase 3 lädt diese on demand.
